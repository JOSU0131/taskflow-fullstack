## Trello
Vamos a Trello, creamos un tablero llamado "HammerFlow Forge"

Para este proyecto utilizo un tablero Kanban en Trello. Divido las tareas en funcionalidades técnicas y muevo las tarjetas según el flujo de trabajo.
